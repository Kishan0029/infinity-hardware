## 🚀 Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed on your machine (if applicable to the stack).

### Installation

1. Clone the repo:
   ```sh
   git clone https://github.com/Kishan0029/infinity-hardware.git
   ```
2. Navigate into the directory:
   ```sh
   cd infinity-hardware
   ```
3. Install NPM packages:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
